##### EyeMS_Server Info


### Agency
POST
- agency/update : 기관 정보 갱신
- agency/delete : 기관 데이터 삭제
- agency/signup : 기관 데이터 저장
- agency/signin : 기관 로그인 정보 일치 확인
GET
- agency/info : 기관 정보 불러오기

### User
POST
- user/signin : 유저 로그인
- user/delete : 유저 데이터 삭제
- user/update : 유저 정보 수정 후 데이터 저장
- user/adduser : 유저 데이터 추가
- user/info : 유저 정보 확인
GET
- user/userlist : 유저 목록 확인

### Unity with User
POST
- info/get : 회원 정보 확인
- info/set : 회원 정보 저장
